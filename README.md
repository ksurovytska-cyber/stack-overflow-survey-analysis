# stack-overflow-survey-analysis
## Project Overview
This project analyzes the Stack Overflow Developer Survey dataset to explore developer profiles, Python usage, remote work trends, work experience, education levels, and compensation by country.

## Technologies
- Python
- Pandas
- Jupyter Notebook / Google Colab
- Descriptive Statistics

## Tasks
- Loaded and explored survey data.
- Checked missing values and completeness of responses.
- Calculated mean, median, and mode for work experience.
- Counted remote workers.
- Calculated the percentage of respondents using Python.
- Analyzed compensation of Python developers by country.
- Identified education levels of the top 5 highest-paid respondents.

## Result
Prepared a structured analytical notebook with tables and insights based on real-world survey data.
